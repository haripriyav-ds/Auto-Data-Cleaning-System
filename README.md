# Auto Data Cleaning & Preprocessing System

## ?? Overview
This project automates data preprocessing tasks such as handling missing values, detecting outliers, and encoding categorical variables.

## ?? Features
- Handles missing values (mean/mode)
- Removes outliers using IQR method
- Encodes categorical variables
- Visualizes data before and after cleaning
- Exports cleaned dataset

## ??? Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib

## ?? Dataset
Titanic Dataset (Kaggle)

## ?? How to Run
1. Download dataset
2. Update file path in code
3. Run the Python script

## ?? Output
- Cleaned dataset saved as CSV
- Graphs showing data distribution
